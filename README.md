# AI Code Reuse Evaluation Platform

AI Code Reuse Evaluation Platform is a web-based system designed to analyze AI-generated code and measure how effectively it reuses existing project logic.

The platform allows users to upload projects, create tasks, run analysis, and view results related to code reuse and AI-assisted development.

## Project Overview

This project is built as a full-stack web application with a Django backend and React frontend.

The main goal of this platform is to help students, developers, and instructors understand how AI tools generate code, how much existing logic is reused, and how the generated code can be improved.

## Main Features

- User registration and login system
- JWT-based authentication
- Student and admin dashboard
- Project upload and management
- Task creation for code analysis
- AI-based code improvement
- Analysis results page
- Clean and responsive user interface
